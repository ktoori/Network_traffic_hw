# Разметка уровня (y = level)

Разметка сделана эвристически, без ручной проверки каждой записи:

1) Если в one-hot признаках должности есть маркеры уровня:
- junior: `junior`, `intern`, `trainee`, `стажер`, `младший`, `начинающий`
- senior: `lead`, `head`, `principal`, `architect`, `team lead`, `tech lead`, `ведущий`, `старший`, `руководитель`, `директор`

То уровень назначается по этим маркерам.

2) Если маркеров нет — используется стаж (`Стаж (месяцев)`):
- <= 24 → junior
- 25..60 → middle
- > 60 → senior

Ограничения разметки:
- название должности не всегда корректно отражает уровень;
- стаж не всегда соответствует уровню;
- часть резюме может быть заполнена неполно.
