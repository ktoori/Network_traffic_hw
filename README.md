# HH.CSV Обработка таблицы

Данный проект в репозитории содержит реализацию пайплайна предобработки табличных данных hh.csv.  
Пайплайн реализован с использованием паттерна проектирования **Chain of Responsibility** и предназначен для подготовки данных к дальнейшему анализу и машинному обучению.

Результатом работы пайплайна являются NumPy-массивы признаков и целевой переменной.

## Описание файлов

**app.py**  
Точка входа приложения. Содержит CLI-интерфейс для запуска пайплайна, сборку цепочки обработчиков и управление процессом выполнения. Отвечает за валидацию входных аргументов и обработку ошибок.

**handlers.py**  
Реализация обработчиков паттерна *Chain of Responsibility*. Каждый класс-обработчик реализует отдельный этап обработки данных (загрузка CSV, очистка, парсинг признаков, кодирование, формирование X и y, сохранение .npy файлов).

**config.py**  
Конфигурационный файл с константами и параметрами пайплайна: название целевой переменной, словари для нормализации данных, шаблоны для one-hot кодирования, списки признаков для снижения кардинальности и другие настройки.

## Использование

```bash
python app.py path/to/hh.csv
```

После выполнения скрипта в директории с исходным CSV-файлом будут созданы:
**x_data.npy** - матрица признаков
**y_data.npy** - целевая переменная
