# HH.csv Pipeline + Regression

Репозиторий содержит на данный момент два задания:

- **`hh_pipeline/`** — preprocessing pipeline (ДЗ1, дедлайн 24.01)  
  Пайплайн загружает сырые данные (CSV), чистит/кодирует признаки и сохраняет:
  - `x_data.npy` — матрица признаков
  - `y_data.npy` — целевая переменная (зарплата, RUB)

  Для удобства проверки HW1 зафиксирована меткой (tag): **`hw1-24-01`**.

- **`hh_regression/`** — обучение и инференс регрессионной модели (ДЗ2, дедлайн 31.01)  
  Модель обучается на `x_data.npy/y_data.npy`, сохраняет веса в `hh_regression/resources/`,
и печатает список предсказанных зарплат (float, RUB).

- **`hh_classification/`** — PoC-классификация уровня разработчика (ДЗ3, дедлайн 17.02)  
  Эксперимент показывает, что по данным hh.ru можно автоматически различать
  уровни **junior / middle / senior** с разумным качеством.


Внутри каждой папки лежит свой подробный README.md с пояснениями и инструкциями по запуску.
