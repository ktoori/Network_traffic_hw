# Постановка задачи

По данным резюме hh.ru необходимо:
1) выделить резюме IT-разработчиков;  
2) сформировать целевую переменную `y` — уровень специалиста: junior / middle / senior (по названию должности и/или опыту);  
3) построить графики баланса классов;  
4) обучить классификатор по признакам резюме;  
5) построить classification report (precision/recall/F1 по классам);  
6) сделать выводы о качестве модели и причинах ошибок.

Проект выполнен как PoC: небольшой эксперимент, подтверждающий, что идея автоматического определения уровня по резюме жизнеспособна.
